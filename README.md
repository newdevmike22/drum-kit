# 🥁 Drum Kit — React Demo App

A sleek and interactive drum kit built with React and powered by Vite. This project focuses on hands-on practice with core React hooks while delivering a responsive, audio-driven user experience.

---

## 🚀 Live Demo

👉 https://funky-drum-kit.netlify.app/

---

## 📌 Overview

**Drum Kit** is a component-based React application that allows users to trigger drum sounds by clicking on visually styled drum pads. Each drum type is mapped to a unique sound, creating an engaging and tactile interface.

The primary goal of this project is to strengthen practical understanding of React hooks and component architecture through repetition and real-world implementation.

---

## 🧠 Key Concepts Practiced

This project emphasizes the use of essential React hooks:

* **useState** — Manages UI state such as active drum pad feedback.
* **useEffect** — Handles side effects like event listeners or lifecycle behavior.
* **useRef** — Provides direct access to DOM elements and audio control.

---

## 🛠️ Tech Stack

* **React** (with Vite)
* **JavaScript (ES6+)**
* **CSS**
* **Font Awesome**
* **AI-generated artwork** (custom background design)

---

## 🎵 Features

* Four interactive drum pads:

  * Crash
  * Kick
  * Snare
  * Tom
* Click-based sound triggering
* Visual feedback on interaction
* Clean and responsive layout
* Custom background artwork for a unique aesthetic

## ⚙️ Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/drum-kit.git

# Navigate into the project directory
cd drum-kit

# Install dependencies
npm install

# Start development server
npm run dev
```
